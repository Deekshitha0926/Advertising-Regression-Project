# 📊 Advertising Regression Project

## 🚀 Overview
This project applies:
- Linear Regression
- Multiple Linear Regression
- Logistic Regression

on the Advertising dataset.

## 📁 Dataset
Contains:
- TV
- Radio
- Newspaper
- Sales

## 🧠 Models
1. Linear Regression (TV → Sales)
2. Multiple Regression (TV + Radio + Newspaper → Sales)
3. Logistic Regression (High vs Low Sales)

## 🔧 Tech Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 📌 Output
- Regression predictions
- Classification results
- Visualizations
